# Sukhpreet Singh — Portfolio (Local Package)

Files included:
- `index.html` — your portfolio webpage
- `Resume_Sukhpreet_Singh.pdf` — your résumé (used by the Download Résumé button)

How to use:
1. Unzip this folder anywhere (e.g., Desktop/Portfolio/).
2. Double‑click `index.html` to open it in your browser.
3. If you update your résumé later, keep the same filename (`Resume_Sukhpreet_Singh.pdf`) or update the link in `index.html`.

Deploy online (optional):
- GitHub Pages/Netlify/Vercel: upload both files keeping their names/paths the same.
